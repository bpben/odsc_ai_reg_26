# ODSC '26 — How I learned to stop worrying and love AI Regulation

Slidev deck for the talk outline (verbatim sections, two-column slides).

## Setup

```bash
cd odsc_ai_reg_26
npm install
```

## Develop

```bash
npm run dev
```

Opens the local Slidev server (default port is shown in the terminal).

## Build static site

```bash
npm run build
```

Output is written to `dist/`.
